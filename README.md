- **Private MCP routing side project** — designing a local MCP router that gives host apps one stable endpoint, routes tool/resource/prompt calls to downstream integrations, manages plugins, and benchmarks routed vs direct execution.
- **Agentic AI infrastructure** — building MCP servers, Agent-to-Agent (A2A JSON-RPC) workflows, and tool-calling integrations for developer-facing platforms.
- **MCP / tool design** — designing schema-driven tools with authentication, session handling, and test coverage for external developer use.
- **Web3 agent identity** — integrating Hedera Agent Kit and ERC-8004-style identity flows into AI-agent workflows.
- **AI development workflow** — maintaining a token-conscious multi-host local agent tooling setup across Claude Code, Codex, OpenCode, Cursor, and Gemini CLI.
